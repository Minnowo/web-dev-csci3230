# User Stories

## Authentication

- As a user, I want to register an account so that my notes are private to me
- As a user, I want to log in and have my session remembered for 7 days so that I don't need to log in every visit
- As a user, I want to log out and have my session cleared so that others can't access my notes
- As a user, I want to be redirected to login after registering so that I can sign in immediately

## Notes

- As a user, I want to create a note so that I can capture my thoughts
- As a user, I want to write and edit note content so that I can update my notes
- As a user, I want to rename a note so that I can organise them clearly
- As a user, I want to delete a note so that I can remove notes I no longer need
- As a user, I want my notes to sync to the backend so that I can access them from any device

## Organisation

- As a user, I want to create folders so that I can group related notes
- As a user, I want to move notes into folders so that my workspace is organised
- As a user, I want to assign an icon to a note so that I can visually distinguish them

## Navigation

- As a user, I want to see my recently visited notes so that I can quickly return to active work

## Visualisations

- As a user, I want to search my notes using keywords so that I can find relevant notes quickly
- As a user, I want to see a graph of how my notes connect so that I can discover relationships
- As a user, I want to see my mood over time so that I can reflect on my writing
